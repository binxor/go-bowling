# Let's Go Bowling

## Overview
Create a 10-pin bowling scoring application that uses traditional scoring rules to score each frame.

## How to Play
Run commands in CLI.
1. Clone this repo
2. Build the app:
```
> go build main.go
```
3. Run the app:
```
> ./main
```
